# SystemSmart
إنشاء النظام بالكامل من الهاتف، يمكنني أن أكون بمثابة "المهندس البرمجي" للمشروع. سنبني SystemSmart خطوة بخطوة داخل GitHub، بحيث في كل مرة أخبرك باسم الملف، ومكانه، ومحتواه الكامل، ثم تنتقل للملف التالي. وعندما تحصل على كمبيوتر لاحقًا، سيكون المشروع منظمًا وجاهزًا للفتح في Visual Studio دون الحاجة لإعادة ترتيبه.
